College Readiness Check


LPL Financial’s 4th Annual University Hackathon-Using AI to Build AI

An advisor-first, AI-assisted prototype that helps families assess college readiness and understand how savings behavior and investment risk affect projected outcomes.



Overview

The tool collects basic planning inputs, projects future tuition and savings, assigns a readiness status, and delivers a clear, risk-aware explanation designed to support advisor-led conversations.



Architecture

Web Client (HTML + Tailwind CSS) — Structured input collection

AWS Lambda (Python) — Projection and status computation

Amazon Bedrock Agent — Governed, compliance-aware explanations

API Response (JSON over HTTPS) — Advisor-aligned insights to client




Principles

Advisor-first • Explainable • Risk-aware • AWS-native • Guardrail-driven
